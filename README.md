# Algoritmo de cálculo de tamanho de diretório em C
## Descrição
Este é um projeto bastante simples que faz uso da função nftw para percorrer 
os diretórios de maneira recursiva, eficiente e sem seguir links simbólicos.
## Exemplo de Uso
### Compilação
```
make
```
### Execução
```
./dirsize <diretorio_alvo>
```
